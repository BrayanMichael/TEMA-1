¿Qué son los Servicios Web?
Los servicios web son componentes de software que permiten el intercambio de información entre sistemas heterogéneos mediante protocolos y formatos estándar (HTTP, JSON, XML). Su adopción facilita la integración, la escalabilidad y la interoperabilidad.
Tipos principales

SOAP

Protocolo basado en XML, con contrato estricto (WSDL). Adecuado para entornos que requieren transacciones y estándares fuertes.

REST / HTTP

Estilo arquitectónico ligero; emplea recursos URL y métodos HTTP; muy usado en aplicaciones modernas.

GraphQL / APIs modernas

Consultas flexibles y eficiencia en la transferencia de datos; ideal para UIs ricas y consumo móvil.

Herramientas y tecnologías
HTML/CSS/JS para interfaces; Postman y cURL para pruebas; Node/Express, Django o Spring para crear servicios; JSON y OpenAPI para documentación.

Pruebas

Postman, Insomnia, pruebas unitarias y contract testing.

Documentación

Swagger / OpenAPI para describir endpoints y contratos.
